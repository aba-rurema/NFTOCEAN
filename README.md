
# NFTOcean

NFTOcean is a decentralized platform that allows users to mint, buy, sell, and trade NFTs. Built with the latest technologies, including **Scaffold-ETH** for smart contract development and **React** for the frontend, it aims to provide a seamless and user-friendly experience for users looking to interact with NFTs.

## Features

* Mint NFTs directly from your browser
* Buy and sell NFTs in a decentralized marketplace
* Support for ERC721 tokens and metadata
* Built using **Scaffold-ETH** for quick Ethereum smart contract deployment
* Responsive UI with React

## Technologies Used

* **Ethereum**: Blockchain network for NFTs.
* **Hardhat**: Ethereum development environment for smart contract deployment.
* **React**: Frontend library for building the user interface.
* **Yarn**: Dependency management for frontend and backend.

## Getting Started

### Prerequisites

Make sure you have the following installed:

* **Node.js** (>= 18.x)
* **npm** (>= 8.x) or **Yarn** (>= 1.x)
* **Git** for version control

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/aba-rurema/NFTOCEAN.git
   cd NFTOCEAN
   ```

2. Install dependencies:

   * Using **npm**:

     ```bash
     npm install
     ```

   * Or using **Yarn**:

     ```bash
     yarn install
     ```

3. Start the development server:

   * Using **npm**:

     ```bash
     npm start
     ```

   * Or using **Yarn**:

     ```bash
     yarn start
     ```

### Running the Application

After following the setup steps, navigate to [http://localhost:3000](http://localhost:3000) to view the application in your browser.
